# Agent HH — агент поиска работы на HH.ru

Десктопное приложение для Windows: поиск вакансий на HH.ru с удобным интерфейсом, предпросмотром объявлений и списком «Отобранные».

## Что умеет

- 8 разделов / 41 профессия (AI-видео, монтаж, операторы, звук, контент, вайбкодинг и др.)
- Фильтры: город, опыт, зарплата от
- Предпросмотр вакансии прямо в приложении
- Кнопка ★ — добавить в «Отобранные» (сохраняется локально)
- Пагинация результатов

## Требования

- [Python 3.9+](https://www.python.org/downloads/)
- [Node.js 18+](https://nodejs.org/)

## Установка

1. Клонировать репозиторий:
   ```
   git clone https://github.com/ВАШ_ЛОГИН/agent-hh.git
   cd agent-hh
   ```

2. Запустить установку:
   ```
   УСТАНОВКА.bat
   ```

## Запуск

```
Agent HH.bat
```

```

│   └── requirements.txt

│   ├── index.html         # UI приложения
│   └── package.json

├── УСТАНОВКА.bat          # Установка зависимостей

```

## Как работает

- `УСТАНОВКА.bat` устанавливает Python-библиотеки (`fastapi`, `uvicorn`, `httpx`) и Electron через `npm install`
- `Agent HH.bat` запускает бэкенд на `127.0.0.1:8000` и открывает окно Electron
- Electron загружает `index.html`, который обращается к бэкенду за вакансиями через [API HH.ru](https://api.hh.ru)
